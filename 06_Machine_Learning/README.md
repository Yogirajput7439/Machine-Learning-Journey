## ✅ 1. Supervised Learning

In supervised learning, data is already labeled and the model learns from input–output pairs.

### Topics and Algorithms Practiced

- Linear Regression
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest
- Naive Bayes
- Support Vector Machine (SVM)

### Tasks Covered

- Classification
- Regression

### Typical Workflow

- Data loading
- Data preprocessing
- Encoding categorical features
- Feature scaling
- Train–test split
- Model training
- Prediction
- Model evaluation

---

## 📊 Evaluation Metrics (Supervised Learning)

### For Classification

- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

### For Regression

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## ✅ 2. Unsupervised Learning

In unsupervised learning, data is not labeled and the goal is to find patterns or structure.

### Topics and Algorithms Practiced

- K-Means Clustering
- Hierarchical Clustering
- DBSCAN
- Dimensionality Reduction (PCA)

### Tasks Covered

- Clustering
- Pattern discovery
- Data grouping
- Visualization using reduced dimensions

### Typical Workflow

- Data preprocessing
- Feature scaling
- Apply clustering / reduction algorithm
- Analyze clusters
- Visualize results

---

## 📊 Evaluation in Unsupervised Learning

- Inertia / Within-cluster sum of squares
- Silhouette Score
- Visual inspection of clusters

---

## ✅ 3. Reinforcement Learning

In reinforcement learning, an agent learns by interacting with an environment and
receiving rewards or penalties.

### Core Concepts Practiced

- Agent
- Environment
- State
- Action
- Reward
- Policy

### Algorithms / Methods

- Q-Learning
- Epsilon-Greedy strategy
- Basic value-based methods

### Typical Workflow

- Define environment
- Initialize agent
- Take action
- Observe reward and next state
- Update policy / Q-values
- Repeat for multiple episodes
