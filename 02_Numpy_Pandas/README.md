## 🔢 NumPy (Numerical Computing)

NumPy is used for handling numerical data and performing fast mathematical operations.

### ✅ Topics Covered

* Arrays (1D, 2D, Multi-dimensional)
* Array Indexing & Slicing
* Reshaping & Flattening
* Mathematical Operations
* Statistical Functions
* Broadcasting
* Random Module
* Linear Algebra Basics

## 🐼 Pandas (Data Analysis & Manipulation)

Pandas is used for data handling, cleaning, and analysis.

### ✅ Topics Covered

* Series & DataFrame
* Data Loading (CSV, Excel, JSON)
* Data Inspection (`head()`, `info()`, `describe()`)
* Data Cleaning

  * Handling Missing Values
  * Filling Null Values (using Mean(), Median(), Mode())
  * Removing Duplicates
* Data Selection & Filtering
* GroupBy Operations
* Merging & Joining DataFrames
* Sorting & Ranking
* Applying Functions
* Time Series Basics
