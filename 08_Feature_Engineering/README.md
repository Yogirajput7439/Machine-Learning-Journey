* Dataset overview

* Missing value inspection
* Basic statistics

### 2️⃣ Univariate Analysis

* Count plots
* Histograms
* Box plots

Purpose:

---

### 3️⃣ Bivariate Analysis

* Feature vs target relationships
* Scatter plots
* Bar plots
* Grouped statistics

Purpose:

### 4️⃣ Multivariate Analysis
